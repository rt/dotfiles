# git-log

*Previous Revisions*

```
:Gclog       //load all previous revisions of the current file into the quickfix list (:cprev :cnext :cfirst :clast)  unimpaired.vim ([q ]q [Q ]Q)
:Gclog -10   //last 10
:Gclog -10 --reverse
:Gclog -1 --until=yesterday
```

*Commits that touched current file*

```
:Gclog -- %  
:Gclog -10 -- % 
```
